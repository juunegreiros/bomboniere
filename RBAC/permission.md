# PERMISSION BOMBONIERE

- `view.product` Visualiza produtos ativos
- `view.productActive` Visualiza produtos ativos (Padrão para customer)
- `edit.product` Edita produtos (Padrão para estoquista)
- `manage.product` Gerencia produtos (criar, editar, ativar/desativar)
- `view.ingredient` Visualiza ingredientes de uma receita
- `edit.ingredient` Edita ingredientes (Padrão para estoquista)
- `manage.ingredient` Gerencia ingredientes (criar, editar, ativar/desativar)
- `view.recipe` Visualiza receita de um produto
- `manage.recipe` Gerencia receitas (criar, editar, ativar/desativar)
- `view.menu` Visualiza cardápio ativo
- `manage.menu` Gerencia cardápios (criar, editar, ativar/desativar)
- `view.menuItem` Visualiza itens ativos do cardápio
- `manage.menuItem` Gerencia itens do cardápio (criar, editar, ativar/desativar)
- `create.order` Criar pedidos, podendo ser pelo customer ou outros roles
- `view.order` Visualiza pedidos feitos (itens, origem)
- `view.ownOrder` Ver meus pedidos (Padrão para customer)
- `edit.orderStatus` Ver status de pedidos
- `manage.order` Gerencia pedidos feitos (criar, editar, histórico de pedidos)
